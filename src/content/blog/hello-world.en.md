This little site opened its doors today.

I've always wanted a corner of the internet that is fully mine — no platform sign-ups, no recommendation algorithm deciding what I see, everything from the words to the look entirely up to me. And here it is: a quiet blog hosted on GitHub Pages, where every post is just a Markdown file.

## What I'll write about

The plan, for now, is three kinds of things:

- **Code**: lessons and pitfalls from tinkering with software
- **Life**: everyday moments worth writing down
- **Video**: if I ever start a YouTube channel, videos will land here too

> Build the site first — the content will grow on its own.

## How it works

Publishing a new post takes three steps: create a Markdown file, write, `git push`. GitHub Actions handles the rest, and a minute or two later the post shows up on this page.

If you'd like to build a site like this one, see [How I Built This Site](/blog/building-this-site/).
