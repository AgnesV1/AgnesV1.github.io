这个小站今天开张了。

一直想有一个完全属于自己的角落——不用注册谁家的平台，不被推荐算法安排，写什么、长什么样都自己说了算。现在它就在这里：一个安安静静的博客，托管在 GitHub Pages 上，每篇文章就是一个 Markdown 文件。

## 以后写什么

暂时的计划是三类：

- **代码**：折腾软件时踩的坑和学到的东西
- **生活**：值得记下来的日常
- **视频**：以后如果开了 YouTube 频道，视频也会同步放到这里

> 先把站建起来，内容会慢慢长出来的。

## 这个站怎么运作

写一篇新文章只需要三步：新建一个 Markdown 文件、写字、`git push`。剩下的交给 GitHub Actions，一两分钟后文章就会出现在这个页面上。

如果你也想搭一个这样的站，可以看看 [How I Built This Site](/blog/building-this-site/)。
